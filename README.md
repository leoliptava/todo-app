# ToDo App

## Link para ver projeto rodando

https://todo.leofelipe.com.br/

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/leoliptava/todo-app.git
```

Entre no diretório do projeto

```bash
  cd todo-app
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
